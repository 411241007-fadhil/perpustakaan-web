# Sistem Informasi Perpustakaan

Aplikasi web sederhana untuk mengelola data perpustakaan kampus menggunakan PHP Native dan MySQL.

## Fitur

- CRUD Data Buku
- CRUD Data Peminjam
- Peminjaman Buku
- Pengembalian Buku
- Pencarian Buku berdasarkan Judul

## Teknologi

- PHP Native
- MySQL
- Bootstrap 5
- HTML5
- CSS3
- JavaScript
- XAMPP
- Visual Studio Code

## Cara Menjalankan

1. Jalankan Apache dan MySQL pada XAMPP.
2. Buat database dengan nama `perpustakaan`.
3. Import file `database/perpustakaan.sql`.
4. Salin folder project ke dalam folder `htdocs`.
5. Buka browser dan akses:

```
http://localhost/perpustakaan-web
```

## Struktur Project

```
perpustakaan-web
│
├── assets
├── config
├── database
├── layouts
├── modules
├── index.php
```
